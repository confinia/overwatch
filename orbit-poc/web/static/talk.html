The architecture is one rule

CelesTrak ─┐               ┌─ MapLibre globe ── reads cache only
SatNOGS  ──┤─►  ingest ─►  Postgres
           │  (the ONLY thing       └─ Grafana ── reads cache only
           │   touching upstream)

Be polite to upstream (CelesTrak bans greedy IPs) — and a Slack front-page spike costs upstream nothing.

Plot twist → the whole point

  • Mid-project, SatNOGS stopped serving decoded values — raw hex only
  • Fix: decode the radio frames locally — 161 community Kaitai decoders
  • "bytes 17–18: battery voltage, ×0.0075 V"

Depending on someone's decoded data = a proxy.
Decoding raw downlinks yourself = infrastructure.
